== Description ==
This plugin creates widget with information about the author of the site. You can also add social links, it will be displayed under the general information.

== Installation ==

1. Upload or extract the `web4pro aboutme` folder to your site's `/wp-content/plugins/` directory. You can also use the *Add new* option found in the *Plugins* menu in WordPress.
2. Enable the plugin from the *Plugins* menu in WordPress.
3. Now the Web4pro About me widget is available.

= Usage =

1. Go to Appearance > Widgets.
2. Enable Web4pro About me widget.
3. Enter widget title
4. Add information to the widget (Name, Caption, Main text).
5. Upload you photo. (We strongly recommend use the "About me" photo size on upload, 50 x 50 pixels).
6. Add necessary links to the social links bar.
7. Click save.
